# lsio

LinuxServer.io (LSIO) repository for GNR-D.

This directory contains configurations, Docker Compose files, and resources related to LinuxServer.io containers and services.

## Structure

- `docker-compose.yml` - Docker Compose configurations for LSIO services
- `configs/` - Service-specific configuration files

## Usage

Place LinuxServer.io Docker container configurations and related resources in this directory.
